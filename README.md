# Claude Code 安装向导 🤖

## ⚡ 太长不看版（1分钟快速上手）

### 这是什么？
一键安装 AI 编程助手的工具，**3步完成**，无需技术基础。

### 怎么用？
**Mac 用户：**
```bash
# 直接双击 m2cc.sh 文件，或在终端输入：
```


1. **运行程序** → 选择 `1`（运行安装向导）

## 🆘 新手故障排除指南

### 🔥 最常见问题快速解决

#### ❌ **问题 1：双击文件没反应**
**可能原因：** Mac 电脑的安全设置
**解决方法：**
1. 右键点击文件 → 选择"打开"
2. 如果还是不行，打开"系统偏好设置" → "安全性与隐私" → 允许运行

#### ❌ **问题 2：提示"命令不存在"**
**可能原因：** 安装后需要重启或刷新环境
**解决方法：**
1. **首选：** 重启终端程序（完全关闭后重新打开）
2. **备用：** 重启电脑
3. **如果还不行：** 重新运行安装程序

#### ❌ **问题 3：网络连接失败**
**可能原因：** 网络不稳定或被防火墙阻挡
**解决方法：**
1. 检查网络连接（打开网页试试）
2. 稍等几分钟后重试
3. 如果是公司网络，可能需要联系 IT 支持

#### ❌ **问题 4：API Key 无效**
**可能原因：** 密钥复制错误或已过期
**解决方法：**
1. 重新复制一次 API Key（确保没有多余空格）
2. 检查密钥是否已过期
3. 重新申请一个新的 API Key

#### ❌ **问题 5：AI 回答很慢或没反应**
**可能原因：** AI 服务繁忙或网络问题
**解决方法：**
1. 检查网络连接
2. 等待几分钟后重试
3. 尝试切换到另一个 AI 模型（推荐使用免费的 GLM-4.5-Flash）

```bash
m2cc
```

然后通过菜单选择所需操作：
- **选项 3**：切换模型提供商
- **选项 4**：查看已配置的模型
- **选项 5**：重新配置模型
- **选项 6**：查看当前状态
- **选项 7**：查看帮助文档

**🎯 新手推荐方式二：直接运行脚本文件**

如果尚未设置全局可执行，可以直接运行：
```bash
./m2cc.sh
```

功能和菜单选项与全局命令完全相同。

### 使用命令行（适合有经验的用户）

如果您设置了全局可执行，推荐使用简洁的命令（无需输入 `./m2cc.sh`）：

```bash
# 显示帮助
m2cc --help

# 查看版本
m2cc --version

# 查看当前配置
m2cc --status

# 重新配置模型
m2cc --configure

# 查看所有已配置的模型
m2cc --list

# 切换模型提供商
m2cc --switch deepseek     # 切换到 DeepSeek
m2cc --switch glm          # 切换到 GLM-4.6
m2cc --switch glm-flash    # 切换到 GLM-4.5-Flash（免费）
m2cc --switch minimax      # 切换到 MiniMax
```

**📝 注意：** 如果尚未设置全局可执行，请使用 `./m2cc.sh` 替代上述命令中的 `m2cc`。

**💡 手动设置全局可执行（如果安装时未设置）：**
```bash
# 复制脚本到系统目录
sudo cp /path/to/m2cc.sh /usr/local/bin/m2cc
sudo chmod +x /usr/local/bin/m2cc

# 之后即可在任何位置使用 m2cc 命令
```
